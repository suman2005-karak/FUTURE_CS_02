## Overview
This repository contains the complete submission for Cyber Security Task 2 under the Future Interns SOC Internship program. The task simulates real-world Security Operations Center (SOC) activities such as monitoring alerts, analyzing logs, and responding to incidents using a SIEM tool.

## Objectives
- Monitor security alerts using SIEM
- Analyze logs to identify suspicious activities
- Classify alerts by severity
- Prepare an Incident Response Report
- Simulate incident communication

## Tools Used
- Splunk SIEM (Free Trial)
- Simulated SOC Logs
- Windows OS
- Web Browser (Splunk Web)

## Key Incidents Identified
- Malware Detection (Ransomware, Rootkit)
- Multiple Failed Login Attempts
- Suspicious External IP Activity
- File Access After Malware Detection

## Conclusion
This project provided hands-on experience in SOC alert monitoring, incident classification, and response planning using Splunk SIEM.
